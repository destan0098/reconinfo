```
git clone https://github.com/destan0098/reconinfo.git
```
## Usage

show IP history , PortScan And Reverse IP
Add Your viewdns.info API Key To api.yaml
```
    reconinfo scan --url example.com --iphistory --portscan --reverse
```
